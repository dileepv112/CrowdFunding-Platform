# How to run the file 
<h2>For client</h2>
<ol>
  <li>cd client</li>
  <li>npm install</li>
  <li>npm run dev</li>
</ol>

<h2>To deploy the contract</h2>
<div>npx thirdweb deploy</div>
<div>For creating thirdweb project:</div>
  <div>npx thirdweb create --app</div>
  <div>npx thirdweb@latest create --contract </div>
